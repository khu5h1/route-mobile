VALIDATIONS

  1. Checks for Password
    1. Username can not contain any special character except underscore(_).
    2. Should not be already present in our database.

FIELD NAMES for general user

  • For User : userid, email_address, home, mobile, fname, lname, pass
  • For Backend : username, email, address, contact, first_name, last_name, password


FIELD NAMES for business user

  • For User : pan_no
  • For Backend : pan_number
